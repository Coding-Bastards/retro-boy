# Retro Boy - Nostalgic Gaming on the Go

Retro Boy is a Mini App that brings on-chain emulation for retro gaming consoles directly to your Pocket Wallet.

## Features

- NFT Catridges: Play games stored as NFTs on the blockchain.
- On-chain save states: Save your game progress directly on-chain.
- Portable gaming: Enjoy classic games on your Pocket Wallet device.
