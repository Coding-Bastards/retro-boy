# Retro Boy - Nostalgic Gaming on the Go

Retro Boy is a Mini App that brings on-chain emulation for retro gaming consoles directly to your phone.

## Core Features

- NFT Cartridges: Play games stored as NFTs on the blockchain.
- On-chain save states: Save your game progress directly on-chain.
- Portable gaming: Keep your games with you wherever you go.
